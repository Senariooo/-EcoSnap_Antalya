#  Teknoloji Yığını (Tech Stack) - EcoSnap

Bu proje, modern ve hızlı bir kullanıcı deneyimi sunmak için aşağıdaki teknolojilerle geliştirilmiştir:

* **Frontend Framework:** React + Vite (Hızlı ve performanslı arayüz).
* **Styling:** Tailwind CSS (Modern ve mobil uyumlu tasarım).
* **Programlama Dili:** TypeScript (Güvenli ve hatasız kod yapısı).
* **AI Model (Yapay Zeka):** Google Gemini 3 Flash (Görüntü işleme, materyal tespiti ve öneri sistemi).
* **Deployment (Yayın):** Netlify (Sürekli entegrasyon ve canlı yayın).
* **Version Control:** GitHub (Kod yönetimi ve sürüm takibi).
* **Geliştirme Aracı:** Lovable AI.
